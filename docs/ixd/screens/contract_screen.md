# 契约管理页

契约 = schema + 订阅 + 版本变更，三个区块在一个页面上。

```
ContractScreen
└── ScrollView
    ├── Section (Schema)
    │   ├── 事件名、来源域、方向、版本
    │   ├── 字段列表（名 / 类型 / 必填）
    │   └── JSON 示例
    ├── Section (订阅方)
    │   ├── 已订阅的域列表
    │   └── Button (订阅此事件)
    └── Section (版本变更)
        ├── Timeline (变更历史)
        └── Button (提交流变更 → Diff 编辑模式)
```

## 提交流变更

点击"提交流变更"后：

```
DiffEditor
├── 当前版本字段列表（可编辑）
├── 变更摘要（自动生成：+2 字段, ~1 字段）
├── 影响范围（列出所有订阅方）
├── Button (发送变更通知)
└── 等待确认
    ├── 订阅方 1 (已确认 ✓)
    ├── 订阅方 2 (待确认 ⏳)
    └── Button (全部确认后发布)
```
