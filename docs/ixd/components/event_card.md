# 事件卡片

## 组件设计

```
EventCard
└── Horizontal (主容器，可点击)
    ├── Vertical (左侧)
    │   ├── Label (事件名：deal.closed，加粗)
    │   ├── Label (来源域：销售云，小字)
    │   └── Label (描述：商机成交时触发，小字，单行截断)
    └── Vertical (右侧，对齐右侧)
        ├── Badge (方向：发布 / 订阅)
        ├── Badge (版本：v2.1)
        └── Badge (订阅方数：3)
```

## 状态

- **正常**：白色背景，标准显示
- **已弃用**：灰色背景，事件名加删除线
- **有变更待确认**：黄色左侧边框指示条
